## 6. 출퇴근 이력 (중요도 30%)    
실제 회사에 가서 일을했다는 증거로 교통카드 내역이나 출퇴근체크 지문기록이 필요한데   
개인지문기록은 없을거고 나는 걸어다녀서 교통카드내역도 없어   
일단 이건 패스할건데 만약 노동청에서 출퇴근 증거가 필요하다고하면 SVN로그 2년치 긁어다가 줄거야
